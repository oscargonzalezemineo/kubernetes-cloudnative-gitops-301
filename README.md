# Kubernetes + Docker Avanzado — CloudNative, GitOps y Observabilidad

[Siguiente página →](labs/M01-entorno-codespace-kind/README.md)

Formación **100 % práctica**: adaptas una aplicación al modelo cloudnative, la despliegas en **Kubernetes** (clúster **kind** en Codespaces), automatizas con **CI/CD** y **GitOps**, y la monitorizas con **Prometheus**, **Grafana** y **ELK**.

## Cómo funciona el curso

1. Haz **fork** de este repositorio en tu cuenta de GitHub.
2. Abre un **Codespace** desde tu fork.
3. Sigue los módulos en orden con **← Página anterior · Siguiente página →**.

Cada módulo tiene **teoría + demostración** en su README y **laboratorios** (`Mxx-NN`) donde practicas tú.

## Antes de empezar

| Requisito | Detalle |
|-----------|---------|
| Cuenta GitHub | Personal y gratuita |
| Navegador | Chrome o Edge (Chromium), actualizado |
| Conexión | Acceso a GitHub, Codespaces, GHCR y Docker Hub sin bloqueos de firewall |
| Equipo | ≥ 8 GB RAM recomendados; no instales Docker ni Kubernetes en local |
| Infraestructura del curso | [infra/README.md](infra/README.md) |
| Documentación de referencia | [docs/](docs/README.md) |
| Problemas frecuentes | [labs/TROUBLESHOOTING.md](labs/TROUBLESHOOTING.md) |

> El bloque **Azure (AKS)** es una **demostración del instructor**. No necesitas suscripción Azure.
